# Modulo3-Semana1
Neste projeto, criei um iterador em Python para a API da tabela FIPE. Importei os módulos requests e json, defini a classe do iterador CarroIterador, obtive dados da API, iterei sobre os dados e imprimi os carros. Este código demonstra a criação de um iterador personalizado em Python para obter e iterar sobre dados de uma API.
